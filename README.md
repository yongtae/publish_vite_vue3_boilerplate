
# vite+vue3 기반 개발프레임워크(publish_vite_vue3_boilerplate)

## 명령어
```
  npm install
  npm run dev
```

## 기반 기능 
```
  axios
  lodash
  mitt
  moment
  pinia
  pinia-plugin-persistedstate
  vee-validate
  vue-codemirror
  vue-router
  vue-skeletor
  vue3-i18n
  vuejs3-logger

```

## 주요 기능 
```
  utils폴더 -> 기반기능 공통모듈화 처리
  layouts폴더-> 레이아웃 기능 처리(헤더영역,푸터영역)
  views -> 화면개발영역
  
```
